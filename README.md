# 🚀 Bias-Aware & Explainable AI Resume Screening System

## 📌 Project Overview
This project implements an intelligent resume screening system using Artificial Intelligence (AI), Machine Learning (ML), and Natural Language Processing (NLP).

The system is designed to:
- Automate resume evaluation
- Reduce human bias in hiring decisions
- Provide explainable insights for transparency

---

## 🎯 Key Features

✅ **Automated Resume Screening**  
Processes resumes and classifies candidates as suitable or not.

✅ **Natural Language Processing (NLP)**  
- Text cleaning  
- Tokenization  
- Feature extraction using TF-IDF  

✅ **Machine Learning Model**  
- Logistic Regression classifier  
- Predicts candidate suitability  

✅ **Bias Mitigation**  
- Removes sensitive attributes (e.g., gender-related words)  
- Promotes fair decision-making  

✅ **Explainable AI (XAI)**  
- Displays important features influencing decisions  
- Improves transparency and trust  

---

## 🧠 Workflow

1. Resume Input  
2. Text Preprocessing  
3. Bias Removal  
4. Feature Extraction (TF-IDF)  
5. Model Training  
6. Prediction  
7. Explainability Output
